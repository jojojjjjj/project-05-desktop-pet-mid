## 一、执行摘要 / Executive Summary

完成本项目需要准备 **10 大类设备**，可归并为 4 大块：

1. **电子焊接与测量**（电烙铁/焊台、热风枪、万用表、镊子、水口钳、PCB 夹具…）—— Track A 主线，Track B 极少；
2. **硬件物料 BOM**（Track A 三块 PCB + 元器件约 ¥120–200；Track B 开发板+舵机约 ¥350–400）+ PCB 打板/SMT/3D 打印制造链；
3. **电脑 + 软件开发环境**（笔记本、ESP-IDF 或 Arduino IDE、VS Code、Python、驱动、Git、云 API 账号）；
4. **工作环境 / 电力 / 网络 / 安全**（焊接台、排插、2.4GHz WiFi、PPE、限流电源、锂电池防火）。

**两档成本**：Track A 最少必需集 ≈ **¥120**（BOM）+ 可复用工具；Track B ≈ **¥378**。均低于 ¥500 红线。

**评审团揪出的 5 个最致命盲点**（Draft 原本遗漏，已补入）：

| # | 盲点 | 后果 | 类别 |
|---|------|------|------|
| 1 | **树脂外壳后处理全套缺失**（IPA 清洗、UV 二次固化箱、拆支撑钳、丁腈手套） | 树脂件发软变形、有毒单体接触皮肤，外壳交付卡死 | 3D 打印 |
| 2 | **FDM 自打配件缺失**（调平纸、铲刀、喷嘴通针、PEI 板） | 首层贴不上/件粘死/堵头，自打路径走不通 | 3D 打印 |
| 3 | **本仓库不含任何 STL 文件**（Track A 在 MakerWorld、Track B 在上游 GitHub），且无 STL 修复工具 | 学员按"从仓库下载"扑空；未修 STL 切片报错/孔位错位 | 3D 打印 |

**红队再补 4 项硬卡点**：限流电源（首次上电防烧板）、锂电池防火（防爆袋+灭火毯/沙，禁水）、关键备件（屏幕排线/电池/模组）、演示日设备（投影/录屏）。

## 十八、最易忽略 Top 10 / Most Overlooked

1. **限流电源**——首次上电防烧板，新手最该有却最少备。
2. **export.bat**——不跑则 `idf.py` 全家 not found，Day1 全员卡死。
3. **pyserial / ffmpeg**——esptool 排错、Track B 音频后端的隐形依赖。
4. **树脂后处理全套**——树脂件不发软变形、不接触有毒单体的必需步骤。
5. **本仓库无 STL**——按"从仓库下载"必扑空，须外部下载+修复工具。
6. **FDM 自打附件**（调平纸/铲刀/通针）——缺则首件即废。
7. **锂电池防火**（防爆袋+灭火毯/沙，禁水）——起火时无应急是重大安全风险。
8. **关键备件**（屏幕排线易坏、电池、ESP32-S3 模组、Type-C 座、FPC 座）——焊废/烧坏无备件则项目停滞。
9. **端口 8081 统一**——SETUP_GUIDE 的 8000 与课程代码的 8081 打架，照抄必 ECONNREFUSED。
10. **GitHub Desktop + 自建仓库 + GitHub Pages**——命令行 git 对高中生门槛极高，且 Demo 网站占 15% 评分必须公网可访问。

## 十九、安全红线 / Safety Red Lines

- **电池**：1.25mm 红黑插头**红正黑负，反接烧充电 IC**；镊子插拔**只捅一边**，同时捅两边短路起火；充电/存放放防爆袋；起火用灭火毯/沙，**禁水**。
- **短路**：每步焊接后万用表蜂鸣档测 VCC-GND；首次上电用**限流电源**；Type-C 16P 焊后必测短路。
- **烫伤**：烙铁 300–350°C，配烙铁架、耐热垫、烫伤急救膏。
- **锡烟/树脂**：焊接通风排风 + KN95；树脂路径加丁腈手套+活性炭面罩+废液收集（IPA 易燃）。
- **插接方向**：核心板↔底板 1.27mm 排针排母**插反烧板**，对照丝印+万用表测短路后再上电。
